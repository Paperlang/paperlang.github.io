Paper Toy

Build paper prototypes and play them over the network.

This project is in development and unreleased. The language interpreter is open source and is available here. Follow the instructions in the readme to build it. To use it in your own project, you can either include it directly (as a subdirectory if you're using CMake) or you can use it as a shared library with the paper.h header file.

Example Snap Game

# Example Game # this game is an implementation of Snap, but with ships # from Eve Online # it's a lot bigger than it has to be game TestGame start  # a visible stack  visiblestack InPlay    # a hidden stack  hiddenstack Draw  # number of players  players 2    card Ship start   int shield   int armour   int hull   int attack  end  # child cards  card Atron Ship start   shield = 300 + 1   armour = 350 * 2 - 1   hull = 400 + 2 * 3   attack = 50 + 60 * 6 - 5 / 4  end    # you dont have to set the attributes  card Kestral Ship start end  card Rifter Ship start end  card Apocolypse Ship start end  card Comet Ship start end  # move random Ship cards into a stack  random Ship -> Draw top 100  # move cards between stacks  Draw -> InPlay top 2  # a setup block, run once at the beggining of the game  setup start   foreachplayer p start    privatestack p.Hand    p.Hand.ownerID = p        random Ship -> Draw top 10    Draw ->_ p.Hand bottom 3   end  end  # a phase declaration, like a function  phase DrawPhase start   Draw -> currentPlayer.Hand top 1  end  phase Place start   # currentPlayer is set while a turn is executing   currentPlayer.Hand -> InPlay top 1   # compare cards from positions in stacks   if InPlay.top == InPlay.top-1 start    # declare a winner    winneris currentPlayer   end  end  # runs every turn  turn start   # run a phase   do DrawPhase   do Place  end end
